# OpenLens

> Capture any region of your screen, extract the text, and do something useful with it — instantly.

OpenLens is a lightweight, cross-platform desktop utility that lets you select a region of your screen, extract text from it via OCR, and interact with that text — translate it, search it, or send it to a local LLM.

---

## Features (Planned — Building Progressively)

- **Screen region capture** — draw a selection over any part of your screen
- **OCR** — extract text from the captured region instantly
- **Floating popup** — clean UI showing the extracted text near your selection
- **Translation** — via Google Translate
- **Browser search** — search selected text on Google or DuckDuckGo
- **Local LLM** — send text to a local model via Ollama
- **Image search** — *(planned)*
- **Offline translation** — *(planned)*
